@startuml
title View Doctor's Schedule
actor User
boundary UI
control Service
database Database

User -> UI: Enter doctor's name keyword
UI -> Service: Search doctors by keyword
activate Service
Service -> Database: Query doctors by keyword
Database --> Service: Return list of doctors
Service --> UI: Return list of doctors
deactivate Service

User -> UI: Select doctor
UI -> Service: Request doctor's schedule
activate Service
Service -> Database: Query schedule by doctor
Database --> Service: Return doctor's schedule
Service --> UI: Return doctor's schedule
deactivate Service

UI --> User: Show doctor's schedule
@enduml
